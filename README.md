# 🚀 OrangeHRM - Automação de Testes com Cypress

Projeto desenvolvido para estudo e prática de automação de testes End-to-End (E2E) utilizando **Cypress** e **JavaScript**, aplicando o padrão de projeto **Page Object Model (POM)** para organização, reutilização e manutenção do código.

Os testes são executados na aplicação **OrangeHRM Demo**, simulando cenários reais de autenticação, navegação e atualização de informações do usuário.

---

## 📚 Tecnologias Utilizadas

- JavaScript (ES6+)
- Cypress
- Node.js
- Page Object Model (POM)
- Git
- GitHub

- ## 🏗️ Padrão de Projeto

Este projeto utiliza o padrão **Page Object Model (POM)**.

Com essa arquitetura, cada página da aplicação possui sua própria classe, contendo:

- Seletores centralizados
- Métodos reutilizáveis
- Separação entre testes e elementos da interface
- Código mais limpo e de fácil manutenção

Essa abordagem facilita a escalabilidade do projeto e reduz a duplicação de código.
